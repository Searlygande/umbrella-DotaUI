⚡ dota-umbrella

umbrella – the main executable that brings all features together. Lightweight, responsive, and built to reduce mechanical overhead in Dota 2.
🎯 Who is this for?

Players who want to spend less time on repetitive input and more time on decision-making. If you've ever wished for cleaner visual feedback or faster access to key items, this might help.
✨ Features

    ⚡ Instant item usage – trigger defensive items and abilities with minimal delay

    📏 Range indicators – see ability distances, power rune timings, and important cooldowns at a glance

    📊 Match context overlay – draft info and relevant stats without tabbing out

    🔄 Input automation – define simple sequences for repeated actions

    🧹 Clean visuals – no clutter, just helpful cues during chaotic fights

    🎮 Custom hotkeys – bind everything to match your preferred layout

🚀 Getting started

    Download the latest version from the Releases page

    Extract the archive to any folder

    Run Loader.exe

    Adjust hotkeys in the settings menu and jump into a match

    💡 The overlay appears automatically in-game. No additional setup required.

💻 System requirements

    OS: Windows 10 or later

    CPU: Intel i3 / AMD equivalent

    RAM: 4 GB or more

    GPU: DirectX 11 compatible

    Storage: ~100 MB free space

⚙️ Configuration

    Open Loader.exe

    Click the gear icon to open settings

    Assign hotkeys for items, abilities, or toggles

    Hit Save – changes apply immediately

All visual helpers are context-aware and adapt to your hero and game state.
